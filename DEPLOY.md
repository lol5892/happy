# Выкладка сайта на хостинг

## Что загружать

Готовая версия для хостинга лежит в папке **`dist/`** или в архиве **`happy-site.zip`**.

Содержимое:

```
dist/
├── index.html
├── styles.css
├── script.js
├── rsvp.php                 ← отправка RSVP в Telegram
├── rsvp-config.example.php  ← шаблон настроек (скопировать на хостинге)
├── robots.txt
├── .htaccess
├── assets/
│   ├── invitation.png
│   └── photos/          ← фото для истории, семьи, героя
└── photos/              ← фото для галереи «Наши любимые моменты»
```

## Как собрать заново

```bash
python _build_gallery.py
python _deploy.py
```

После этого обновятся `dist/` и `happy-site.zip`.

## Как залить на хостинг

1. Откройте панель хостинга (Timeweb, Beget, Reg.ru и т.п.).
2. Перейдите в **Файловый менеджер** → папку сайта (`public_html`, `www` или `htdocs`).
3. Удалите стандартный `index.html`, если он там есть.
4. Загрузите **все содержимое** папки `dist/` (не саму папку, а файлы из неё).
5. Откройте сайт в браузере и проверьте:
   - главную страницу;
   - историю с каруселями;
   - галерею;
   - кнопку вишлиста;
   - обратный отсчёт.

## Уведомления в Telegram (RSVP)

После отправки формы «Подтверди, пожалуйста» вам придёт сообщение в Telegram с именем гостя и ответом.

### Настройка (один раз)

1. **Создайте бота** в Telegram: напишите [@BotFather](https://t.me/BotFather) → `/newbot` → придумайте имя и username.
2. **Скопируйте токен** бота (формат `123456789:AAH...`).
3. **Узнайте chat_id**:
   - Напишите боту `/start` (или любое сообщение).
   - Откройте в браузере:  
     `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
   - Найдите `"chat":{"id":123456789}` — это ваш `chat_id`.
4. **Для двух получателей** (Антон и Таня): каждый пишет боту `/start`, добавьте оба `chat_id` в массив.
5. **Для общего чата/группы**: добавьте бота в группу, напишите там сообщение, возьмите `chat_id` из `getUpdates` (у групп id отрицательный, например `-1001234567890`).
6. На хостинге скопируйте `rsvp-config.example.php` → **`rsvp-config.php`**.
7. Вставьте в `rsvp-config.php`:
   - `telegram_bot_token` — токен из шага 2;
   - `telegram_chat_ids` — массив id, например `[123456789, 987654321]`.

### Проверка

Откройте сайт, заполните форму RSVP и нажмите «Буду!» или «Не смогу». В Telegram должно прийти сообщение вида:

```
🎂 RSVP · Мирон 1 годик

Имя: бабушка Лена
Ответ: Буду! 🎉
Гостей: 2
Время: 08.06.2026 15:30
```

> Гостю всегда показывается подтверждение на сайте, даже если уведомление в Telegram временно не отправилось.

## Важно

- Для уведомлений в Telegram на хостинге нужен **PHP** (есть на Timeweb, Beget, Reg.ru и др.).
- Размер сайта около **200+ МБ** из‑за фотографий — загрузка может занять несколько минут.
- Папки `photos/` (исходники) и `шаблон/` на хостинг **не загружать** — они только для редактирования у себя на компьютере.
- Файл **`rsvp-config.php`** с токеном не публикуйте — создайте его только на хостинге.

## Если что-то не работает

- **Галерея пустая** — запустите `python _build_gallery.py`, затем `python _deploy.py`.
- **Фото не открываются** — проверьте, что папки `assets/photos/` и `photos/` загружены полностью.
- **Кодировка** — файл `.htaccess` уже включён для Apache-хостинга.
- **В Telegram не приходит RSVP** — проверьте `rsvp-config.php`, токен бота, `chat_id` и что вы написали боту `/start`.
