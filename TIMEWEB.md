# Сайт для гостей в России (без VPN)

## В чём проблема

`miroshegodik.ru` сейчас идёт через **Cloudflare** (оранжевое облако) → серверы Cloudflare Pages.

В России IP-адреса Cloudflare часто **блокируют или режут скорость**. Поэтому:

- без VPN сайт может **не открываться**;
- с VPN — открывается, но **медленно** и фото могут **не догружаться**;
- `*.pages.dev` — та же сеть Cloudflare, та же история.

Это **не ошибка DNS** и не баг сайта. Для гостей в РФ нужен **российский хостинг**.

## Решение: Timeweb (домен у вас уже там)

### 1. Хостинг

В панели Timeweb закажите **виртуальный хостинг** (если ещё нет) и привяжите домен `miroshegodik.ru`.

Запомните **IP-адрес сервера** (в разделе «Сайты» / «Информация о сервере»).

### 2. Собрать сайт на компьютере

```bash
python _build_gallery.py
python _deploy.py
```

Появятся папка `dist/` и архив `happy-site.zip` (~200 МБ).

### 3. Загрузить на Timeweb

Файловый менеджер → папка сайта (`public_html` или `www`):

1. Удалите стандартный `index.html`, если есть.
2. Загрузите **всё содержимое** `dist/` (не саму папку, а файлы внутри).
3. Скопируйте `rsvp-config.example.php` → `rsvp-config.php` и вставьте токен бота и chat_id (как в `DEPLOY.md`).

### 4. Переключить DNS (важно!)

Нужно, чтобы домен вёл **на Timeweb**, а **не** на Cloudflare Pages.

#### Вариант А — NS остаются на Cloudflare

1. Cloudflare → **DNS** → удалите CNAME `miroshegodik.ru` → `happy-c3c.pages.dev`.
2. Добавьте **A-запись**:
   - Имя: `@` (или `miroshegodik.ru`)
   - IP: **IP вашего хостинга Timeweb**
   - Прокси: **DNS only** (серое облако, не оранжевое!)
3. По желанию: `www` → CNAME на `miroshegodik.ru` (тоже серое облако).

#### Вариант Б — вернуть NS на Timeweb

1. В Timeweb в DNS домена: **A** `@` → IP хостинга.
2. В регистраторе домена смените NS с Cloudflare обратно на NS Timeweb.

### 5. Проверка

Откройте `https://miroshegodik.ru` **без VPN** с телефона на мобильном интернете:

- главная и история;
- фото в галерее;
- форма RSVP.

### 6. Cloudflare Pages

Проект `happy` на Cloudflare можно **оставить** как резерв (ссылка `happy-c3c.pages.dev` с VPN). Для гостей в России отправляйте **`miroshegodik.ru`** после переезда на Timeweb.

## Обновление сайта после правок

1. Правки в проекте → `python _build_gallery.py` → `python _deploy.py`.
2. Залить обновлённые файлы из `dist/` на Timeweb (или только изменённые).

## RSVP

| Где сайт | Как работает форма |
|----------|-------------------|
| Cloudflare Pages | `/api/rsvp` + переменные в Cloudflare |
| Timeweb (PHP) | `rsvp.php` + `rsvp-config.php` на сервере |

Скрипт `_deploy.py` сам подменяет endpoint на `rsvp.php` в сборке для хостинга.
